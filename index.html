about me

in 2015 I moved from California to South Carolina to enroll in the Iron Yard; a code bootcamp where I learned the fundamentals of front end web development, as well as "learning how to learn". I chose front end web development because it suited my creativity better than the more technical backend web development. In fact, my focus on design as well as code is what I take the most pride in, and lead me to my first employment opportunities. My focus when creating for the web, is to approach a site with three main goals

  • palette
    design

    using industy leading tools, the design must stand out while also provide a seemless user experience

  • laptop
    build

    combining CSS, Javascript, and the advandages of HTML5, sites are built cross browser compliant, and seo friendly

  • phone_iphone
    responsive

    Using a mobile first focus allows websites to work with any viewport. My designs focus on reaching as many devices as possible.

I'm always learning new things to add to my skill sets; whether it be new design focuses like improved typography and use of space, or getting to know new technical tools like REACT. As of 2021, I'm open for work.
